# xml-releve

Ce projet contient un **document XML** représentant un **relevé bancaire**.  
Chaque relevé est associé à un **RIB**, une **date**, un **solde**, et un ensemble d’**opérations**.  

Chaque opération contient :  
- un **type** : CREDIT ou DEBIT  
- une **date**  
- un **montant**  
- une **description**
